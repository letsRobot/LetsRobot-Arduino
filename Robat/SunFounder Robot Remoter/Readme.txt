SunFounder Robot Remoter.exe For Microsoft Windows

It is applicable to SunFounder Crawling Quadruped Robot Kit for Arduino.
You must install Microsoft .NET Framework 4.0 or higher version.
If you have not install it, run dotNetFx40_Full_setup.exe.

How to use
1. Connect your remote controller to PC, run SunFounder Robot Remoter.exe.
   The program will automatically select the serial port and establish a connection.
   If the default serial port is wrong, reselect another one.
2. Click the button on the program interface, or use your keyboard to control the robot.
